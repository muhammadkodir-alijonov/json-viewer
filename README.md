# JSON Viewer — Full Stack

Real-time collaborative JSON viewer/formatter.

## Ishga tushirish

### 1. Backend
```bash
cd backend
npm install
npm run dev
# → http://localhost:3001
```

### 2. Frontend (yangi terminal)
```bash
cd frontend
npm install
npm run dev
# → http://localhost:3000
```

## Stack
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, Monaco Editor, Yjs, Socket.io-client
- **Backend:** Node.js, Express, Socket.io, Yjs, TypeScript
